# 仿掘金官网

## 运行教程

### 前端（imitate-juejin文件夹）
```
// 前端目录
cd juejin

// 安装依赖
npm install

// 启动服务
npm run serve
```
#### 前端本地URL

本地：http://localhost:8080

### 后端（imitate-juejin-java文件夹）
```
// 前端目录
cd juejin

// 安装依赖
npm install

// 启动服务
npm run serve
```
